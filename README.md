## HOW TO BUILD
1. download the source code with git:
```
git clone https://github.com/FrogPossibility/CobraConfig.git
cd CobraConfig
```

2. install the dependencies:
```
npm install
```

3. test it:
```
node main.js
```

3. build it:
```
npm run build
```
